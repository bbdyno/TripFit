# TripFit 개인정보 처리방침

시행일: 2026-02-20

TripFit("당사", "우리")는 사용자의 개인정보를 중요하게 생각합니다. 본 개인정보 처리방침은 TripFit에서 정보를 어떻게 처리하는지 설명합니다.

## 1. 수집하는 데이터

TripFit은 오프라인 우선(offline-first) 앱이며, 계정 생성 없이 사용할 수 있습니다.

사용 방식에 따라 아래 데이터가 저장될 수 있습니다.
- 옷장 데이터(아이템명, 카테고리, 컬러, 태그, 메모)
- 코디/여행 계획 데이터(이름, 날짜, 목적지, 메모)
- 사용자가 추가한 의류 사진(카메라/앨범)
- 앱 환경설정(테마, UI 옵션, 동기화 설정)

## 2. 데이터 저장 방식

- 기본적으로 데이터는 기기 로컬에 저장됩니다.
- iCloud 동기화를 켜면 Apple CloudKit/iCloud를 통해 기기 간 동기화가 진행됩니다.
- 동기화 설정의 "Include Wardrobe Photos"가 켜져 있으면 의류 사진도 iCloud 동기화 대상에 포함됩니다.

## 3. 데이터 이용 목적

수집된 데이터는 아래 기능 제공을 위해서만 사용됩니다.
- 옷장/코디/여행 기능 제공
- 기기 간 동기화(사용자가 켠 경우)
- 사용자 본인 Apple 기기에서 데이터 복원

당사는 개인정보를 판매하지 않습니다.

## 4. 제3자 제공

광고사 또는 데이터 브로커에게 사용자 데이터를 제공하지 않습니다.

iCloud 동기화 사용 시 데이터 처리는 Apple의 iCloud/CloudKit 약관 및 개인정보 정책이 적용됩니다.

## 5. 보관 기간 및 삭제

- 로컬 데이터는 사용자가 삭제하거나 앱을 제거할 때까지 기기에 남아 있습니다.
- iCloud 동기화 데이터는 앱 또는 iCloud에서 삭제하기 전까지 유지될 수 있습니다.
- 앱 내 초기화 기능 실행 시 로컬 및/또는 iCloud 데이터가 영구 삭제될 수 있습니다.

## 6. 보안

당사는 Apple 플랫폼 보안 메커니즘과 합리적인 보호 조치를 사용합니다. 다만 어떤 저장/전송 방식도 100% 안전하다고 보장할 수는 없습니다.

## 7. 아동 개인정보

TripFit은 만 13세 미만 아동을 대상으로 하지 않으며, 해당 아동의 개인정보를 고의로 수집하지 않습니다.

본 방침은 필요 시 업데이트될 수 있으며, 최신 버전은 본 문서에 게시됩니다.
